# Управление профилем

## Описание

Добавлена функция редактирования профиля для ресторанов и поставщиков. Каждый пользователь может обновить информацию о своей компании и ИНН.

## Функциональность

### Для ресторана
- Редактирование названия ресторана
- Редактирование ИНН ресторана
- Редактирование номера телефона
- Просмотр email (не изменяется)

**Маршрут:** `/restaurant/profile`

### Для поставщика
- Редактирование названия компании
- Редактирование ИНН компании
- Редактирование номера телефона
- Просмотр email (не изменяется)

**Маршрут:** `/supplier/profile`

## Реализация

### Схема базы данных
Добавлены новые поля в таблицу `users`:
- `restaurantInn` (опционально) - ИНН ресторана
- `supplierInn` (опционально) - ИНН поставщика

### API мутации
- `updateRestaurantProfile` - обновление профиля ресторана
- `updateSupplierProfile` - обновление профиля поставщика

### UI компоненты
- `/restaurant/profile` - страница профиля ресторана
- `/supplier/profile` - страница профиля поставщика
- Добавлен пункт "Профиль" в меню сайдбара для обоих типов пользователей

## Использование

1. Авторизуйтесь как ресторан или поставщик
2. Нажмите на "Профиль" в меню сайдбара
3. Нажмите "Редактировать профиль"
4. Заполните необходимые поля
5. Нажмите "Сохранить"

## Защита маршрутов

Оба маршрута защищены middleware и требуют авторизации. Неавторизованные пользователи будут перенаправлены на страницу входа.
